ダンボールでRaspberry Piのケースを作った

材料

  • amazonのダンボール
  • 瞬間接着剤
  • 被覆された針金的なもの
    • 頭と胴体とかをつなぐため

サイズ[cm]

  • 頭: w10 x h7 x d7
    • wifiのusbドングルを使うことを考えると、w11の方がよかった。
    • SDカードもはみ出さないやつを使う必要がある
  • 胴体: w5 x h5.5 x d5
  • 手: w2 x h7.5 x d2
  • 足: w2 x h4 x d4.5

顔がでかい気がするので、他のパーツをもう少し大きくしてもいいかもしれない
