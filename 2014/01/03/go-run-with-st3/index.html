sublime text 3でgolangを実行(go run)する

Tools > Build System > New Build System... でこんな感じに設定する。

  • Go.sublime-build
{
  // "cmd": ["go", "run", $file], // for sublime text 2
  "shell_cmd": "go run $file",
  "selector": "source.go",
}

もしくは、DisposaBoy/GoSublimeをインストールする。

これを入れておくとファイル保存時にgo fmtを自動的に実行してくれたり、
いい感じに入力補完もしてくれて便利。
