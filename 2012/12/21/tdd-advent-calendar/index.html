今年の9月にTDDBC横浜 Second Seasonに参加した
ものの、
その後テストを書かない日の方が多かったので、
これはまずいと思ってテストを書き続けられるようなツールを作りました。

fukayatsu/testing-code-bot · GitHub

(Readmeとかまだ無いです。ごめんなさい)

「テスト書いた #testingcode」などとつぶやくと、
プロフィール写真をgreenな感じに変えてくれます。

逆に、greenの状態のままなにもしないで一定時間(とりあえず36時間)経過すると、
プロフィールアイコンをredな感じに変えてくれます。

redな感じになっていたら恥ずかしいのできっとテストを書き続けられるはずです。

来年はずっとグリーンでいられるように頑張りたいと思います！
