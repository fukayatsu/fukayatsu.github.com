サンプルコード(一応写経)

```go
// hello.go
package main

import "fmt"

func main() {
  fmt.Println("Hello, 世界")
}
```

実行してみる
```
% go run hello.go
Hello, 世界
```

ビルドして実行してみる
```
% go build hello.go
% chmod +x hello
% ./hello
Hello, 世界
```
